## Overview

OpenCowork is an open-source agentic workspace that enables autonomous execution of multi-step tasks.

### Key Components

- **Agent Planner**: Converts goals to execution plans
- **Task Executor**: Runs plans step-by-step
- **Tool Layer**: Interface to system operations
- **Sandbox**: Isolated execution environment
- **Desktop UI**: User-facing application
